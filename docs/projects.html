FINAL YEAR GAME

In my final year, I developed a 2D action-platformer game in Unity, themed around medieval fantasy, featuring responsive player controls and enemy AI using state-driven logic. I designed modular systems using C# and key design patterns (Singleton, Factory, State) to support features including a stat upgrade system and dynamic achievement tracking. I Gained experience in animation syncing, scene transitions, and architecting scalable UI across multiple gameplay states.
